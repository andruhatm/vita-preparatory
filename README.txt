Данное программное решение позволяет расчитать: может ли шар докатиться до выхода и колво требуемых ходов и визуализировать процесс посредством библиотеки Swing.

Для получения входной информации используются 4 файла, примеры файлов прилагаются
board_data.txt - информация о типе клетки поля (1 - закрывающаяся перегородка, 2 - вход, 3 - выход, 4 - открывающаяся перегородка)
obstacles.txt - периоды открытия/закрытия перегородок
turn.txt - номер хода с которого начинается симуляция
heights.txt - информация о высоте клетки, используется для свободных тайлов и перегородок в момент открытия
Симуляция каждого шага производится при нажатии на соответствующую кнопку.
При блокировке камня обьект принимает красный цвет, при тупиковой ситуации зеленый.

Результат расчетов программы выводится в файл "D:/result.txt"
